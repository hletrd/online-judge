# 02. 대회 시스템 (v3 — 정정판)

## 총폐 (★★★★☆)

v2는 ★★★이었으나, 클랜(Q&A) 시스템, 대회 공지, 대회 리플레이, 프리즈 중 본인 순위 확인, 가상 대회가 **모두 완전 구현**되어 있음이 확인되어 ★★★★로 상향 조정한다. ICPC/IOI 스코어링, 프리즈, 클랜, 공지, 리플레이, 가상 참여, 리더보드 시각화 — 대회 운영의 핵심 기능이 거의 다 갖춰져 있다.

---

## 랭킹 & 리더보드 (★★★★★)

### 장점

**시각적 퀄리티가 높다.** (`contest/leaderboard-table.tsx`)
- Top 3 포디움 표시 — Codeforces나 BOJ 대회보다 시각적으로 우수
- ICPC 모드: 페널티 표시 (+ 표기), AC/WA/대기 색상 코딩
- IOI 모드: 점수 그라디언트 표시 (0%→100% 빨강→초록), 다크모드 인식 색상
- **프리즈 기능** — 파란 테두리 + snowflake 배지 + "Frozen" 레이블
- **프리즈 중 본인 실시간 순위** — "Live Rank" 배지로 본인만 실제 순위 표시
- 내 순위 하이라이트 (ring + 배경색)
- 30초 자동 갱신 (탭 숨김 시 일시정지)

v2가 "프리즈 중 본인 순위 확인 불명확"이라고 했으나, **Live Rank 배지로 구현되어 있다.**

### BOJ 대회 리더보드와의 비교

JudgeKit의 리더보드는 **BOJ 대회 리더보드보다 명백히 우수**하다. BOJ는 기능적으로 최소한이고 시각적으로도 구식이다. JudgeKit은 현대적이고 정보 밀도가 높다.

## 클랜(Clarification/Q&A) 시스템 (★★★★☆)

### 실제 구현 상태 (v2 정정)

**클랜 시스템이 완전 구현되어 있다!** `contest-clarifications.tsx`:

- **참가자**: 문제별 또는 "일반" 질문 작성
- **출제자/운영자**: Yes/No/No Comment/Custom 답변
- **공개/비공개 토글**: 운영자가 질문의 공개 범위 제어
- **삭제 기능**: 운영자만
- **자동 갱신**: 30초 주기 (탭 숨김 시 일시정지)
- **상태 표시**: answered/pending 배지

이것은 **Codeforces의 Q&A 시스템과 동등한 수준**이다. v2가 "클랜 시스템이 없다 — CRITICAL, 없으면 실제 대회 운영 불가"라고 한 것은 **가장 심각한 사실 오류**였다. 대회 운영에 필요한 가장 기본적인 기능이 이미 작동하고 있다.

## ICPC vs IOI 스코어링

### ICPC 모드
- 구현됨: 페널티 기반 순위, 프리즈, Live Rank
- **누락**: 풍선(Balloon) 표시 — 문화적 요소이지 기능적 결함은 아님
- **누락**: 팀 대회 (별도 항목)

### IOI 모드
- 구현됨: 점수 그라디언트, 서브태스크 개념(부분 점수)
- **누락**: 서브태스크 점수 구조 — 현재는 테스트케이스별 점수만 있고, 그룹/서브태스크 단위 점수 부여가 안 됨
- **누락**: 점수 변화 그래프

---

## 여전히 누락된 핵심 기능

**1. 팀 대회 미지원** (P1)
ICPC는 3명이 한 팀으로 참가. 팀 엔티티, 공동 제출, 팀 랭킹이 필요.

**2. 서브태스크 / 그룹 채점** (P0)
IOI 모드에서 "조건 A → 30점, B → 70점, C → 100점" 구조. 현재 테스트케이스에 그룹 ID/점수가 없다.

**3. 대회 중 문제 잠금 해제(Problem Unlock)** (P2)
일부 포맷에서 문제 순차 공개. IOI 스타일.

**4. 대회 아카이브가 약하다** (P2)
과거 대회의 문제 세트, 제출, 순위를 모아보는 아카이브.

**5. 대회별 통계** (P2)
문제별 정답률, 참가자 수, 분포도.

## 개선 제안 (우선순위 순)

1. **서브태스크 / 그룹 채점** — IOI 모드를 제대로 지원하려면 필수
2. **팀 대회 지원** — ICPC 목적이면 필수
3. **대회 아카이브 & 통계** — 대회 후 분석용
4. **문제 공개 제어** — 순차 공개, 조건부 공개
5. **서브태스크 점수 시각화** — IOI 모드 강화
